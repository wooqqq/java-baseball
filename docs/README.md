## 구현 사항
### 시행 순서
1. 사용자가 숫자 입력

        숫자 야구 게임을 시작합니다.
        숫자를 입력해주세요 : [입력값]
2. 입력한 숫자와 답안을 계산하여 결과값 출력

        1볼 1스트라이크
3. `3스트라이크`일 시 게임 종료

        3스트라이크
        3개의 숫자를 모두 맞히셨습니다!
4. 새 게임 시작 유무를 입력받음

        게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.
        [입력값]


### 예외처리
- 입력받은 숫자
  - 서로 다른 3자리의 수
  - 숫자 형태의 입력값
  - 빈칸이 포함될 수 없다
- 게임이 끝난 경우 재시작/종료를 구분하는 숫자
  - 1 또는 2가 입력되어야 한다
  - 빈칸이 포함될 수 없다
